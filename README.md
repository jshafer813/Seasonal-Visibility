## Installation

### Via plugin repository (recommended)
1. Go to **Dashboard → Plugins → Repositories** and add:
```
https://raw.githubusercontent.com/jshafer813/Seasonal-Visibility/main/manifest.json
```
2. Go to **Catalog**, find **Seasonal Visibility**, and install it.
3. Restart Jellyfin, then open **Dashboard → Plugins → Seasonal Visibility**.

### Manual
1. Download `Jellyfin.Plugin.SeasonalVisibility.dll` from the [latest release](https://github.com/jshafer813/Seasonal-Visibility/releases/latest).
2. Create `<data-dir>/plugins/Seasonal Visibility_1.0.2.0/` and place the DLL inside it.
3. Restart Jellyfin, then open **Dashboard → Plugins → Seasonal Visibility**.
